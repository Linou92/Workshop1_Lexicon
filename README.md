Markdown is a lightweight markup language for creating formatted text using a plain-text editor. This guide covers common Markdown elements and how to use them.

## **Headers**
Markdown uses `#` symbols for headers:
```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
```
**Example:**
# H1
## H2
### H3
